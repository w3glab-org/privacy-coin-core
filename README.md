# Privacy Coin Core

Privacy pool for ETH transactions using zero-knowledge proofs.

## Setup

```bash
yarn install
yarn compile
```

## Environment

Create a `.env` file:

```
PRIVATE_KEY=<deployer private key>
ALCHEMY_KEY=<alchemy api key>
ETHERSCAN_KEY=<etherscan/basescan api key>
ADMIN_ADDRESS=<admin/multisig address (optional, defaults to 0x44eb9939cfdE7C394f1632C6890191d695f0a3ce)>
```

## Tests

```bash
yarn test
```

## Deploy

```bash
# Compile (always compile again before deployment!)
yarn compile

# Base (mainnet Ether)
npx hardhat run scripts/deploy.js --network base

# Base (mainnet USDC)
npx hardhat run scripts/deployERC.js --network base

# Ethereum (mainnet Ether)
npx hardhat run scripts/deploy.js --network mainnet

# Ethereum (mainnet USDT)
npx hardhat run scripts/deployERC.js --network mainnet
```

## Upgrade

Deploy a new implementation and upgrade the proxy:

```bash
PROXY_ADDRESS=<proxy address> npx hardhat run scripts/deployUpgrade.js --network baseSepolia
```

Go to Safe transaction builder:
To Address: <proxy address>
Eth Value: 0
Contract Method Selector: upgradeToAndCall
newImplementation (address): implementation address deployed from scripts/deployUpgrade.js script
data: 0x

## Architecture

- **EtherPool** -- Main contract handling deposits, withdrawals, and shielded ETH transfers with ZK proof verification. Inherits from `MerkleTreeWithHistory` and `UUPSUpgradeable`.
- **ERCPool** -- Main contract handling deposits, withdrawals, and shielded USDC transfers with ZK proof verification. Inherits from `MerkleTreeWithHistory` and `UUPSUpgradeable`.
- **MerkleTreeWithHistory** -- Poseidon-based Merkle tree with a circular root history buffer (100 roots).
- **Verifier2** -- Groth16 ZK proof verifier (generated by snarkjs).
- **Hasher** -- On-chain Poseidon. Bytecode is **generated** by `circomlibjs` in `scripts/compileHasher.js`. There is no Solidity source in `contracts/` for this blob, so normal explorer **Solidity** verification doesn't work. Trailing bytes of `abis/Hasher.abi.json` should match onchain Hasher contract bytes.
- **Proxy** -- ERC-1967 UUPS proxy. Upgrade authority is controlled by the contract's `admin`.
